dezyre-nyse
===========

NYSE BigData project

Setup single node Hadoop on Mac
================================

1. install brew
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

2. install hadoop, hive, zookeeper, sqoop
```
$ brew install hadoop
$ brew install zookeeper
$ brew link zookeeper
$ brew install sqoop
```
