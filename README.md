dezyre-nyse
===========

NYSE BigData project

Setup Single node cluster on Mac
================================

1. install brew
```ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"```
2. install hadoop, hive, zookeeper, sqoop
```
$ brew install hadoop
$ brew install zookeeper
$ brew link zookeeper
$ brew install sqoop
```
